# DevOps

**Termin wykonania zadania: 06.11 23:59**

Zaproponuj diagram infrastruktury w chmurze (najlepiej AWS lub GCP) dla produktu składającego się z aplikacji webowej i mobilnej.

Specyfikacja produktu:
- Wspólna baza danych PostgreSQL
- Wykorzystuje zewnętrzne API chronione kluczem
- Do przechowywania pamięci podręcznej aplikacji wykorzystywany jest Redis.
- Aplikacja przechowuje pliki udostępniane przez użytkowników.

Zadbaj o to, aby infrastruktura przedstawiona na diagramie oparta była o dwie strefy dostępności (Availability Zone).
